# hidalgo-design-token-system

# Design Token System — App Checador  Este repositorio contiene la fuente de verdad ("Single Source of Truth") para el diseño visual de la **App Checador**. Centraliza colores, tipografía, espaciados y reglas de UI para garantizar una experiencia de usuario consistente entre el portal web y la aplicación móvil.  ## Colores Institucionales - **Primario:** #A02142 (Guinda institucional) - **Secundario:** #BC955B (Dorado institucional) - **Acento/Terciario:** #6F7271 (Gris institucional)  ## Contenido - `tokens.json`: Definición base de los tokens en formato crudo. - `tokens.web.ts`: Adaptación de los tokens para ser consumidos por **Tailwind CSS** y **shadcn/ui** en el repo `checador-web`. - `tokens.native.ts`: Adaptación de los tokens para **React Native** (Expo) en el repo `checador-mobile`.  ## Uso  ### Web (Next.js) Importa `tokensWeb` en tu configuración de Tailwind para extender el tema: ```typescript import { tokensWeb } from '../design-tokens/tokens.web'; // ... usar en tailwind.config.ts ```  ### Mobile (React Native) Importa `tokensNative` para usar en tus `StyleSheet`: ```typescript import { tokensNative } from '../design-tokens/tokens.native'; // ... style={{ color: tokensNative.colors.brand.primary }} ```  ## Mantenimiento Cualquier cambio en la identidad visual debe realizarse primero en `tokens.json` y posteriormente propagarse a los archivos `.ts` específicos.
